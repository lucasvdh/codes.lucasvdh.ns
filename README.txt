Bring Dutch Railways (NS) into your Homey. Pin a station or a route as a device and use Flows to react to delays, cancellations, disruptions, track changes and busy trains. Plan trips, fetch the next departure or arrival, and log delays and crowdedness in Insights.

This app uses the official NS public APIs and needs a free personal API key. Setup walkthrough and full documentation:

https://homey-nederlandse-spoorwegen.gitbook.io/docs-en
