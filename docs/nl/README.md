# Welkom bij NS voor Homey

**De NS-app voor Homey maakt van Nederlandse treindata iets waar je huis op kan reageren.** Pin een station of een dagelijks traject als Homey-apparaat, automatiseer flows op vertrektijden, vertragingen, uitval, drukte of storingen, en lees alles terug in Insights.

De app praat met de openbare NS API's ([Reisinformatie API](https://apiportal.ns.nl/api-details#api=reisinformatie-api) en [Disruptions API](https://apiportal.ns.nl/api-details#api=Disruptions-Api)) met je eigen subscription key — je data blijft van jou.

## Wat je ermee kunt

- Een melding 5 minuten voor je laatste trein 's avonds, zodat je hem niet mist.
- Het licht aan zetten zodra je trein naar huis vertrekt en dimmen als hij het volgende station passeert.
- Je speakers laten omroepen dat de 8:19 naar Utrecht op spoor 5b in plaats van 5a komt.
- Een pushbericht als er een storing is op je gebruikelijke woon-werkroute.
- Vertraging-in-minuten per traject loggen en in Homey Insights visualiseren.

## Disclaimer

Dit is een onofficiële, externe Homey-app. De app is niet gelieerd aan, ondersteund door of gesponsord door NS Groep N.V. Treindata komt van de openbare NS API die "as is" wordt aangeboden — accuraatheid en beschikbaarheid liggen buiten de controle van deze app.
